# Tweeter Project

A simple single-page AJAX Twitter clone that uses JQuery, HTML5 and CSS to help students get more comfortable with front-end development.

## Dependencies

"body-parser": "^1.15.2",
"chance": "^1.0.2",
"express": "^4.13.4",
"md5": "^2.1.0",
"timeago.js": "^4.0.2"
